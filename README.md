## Cài đặt và triển khai Hadoop single node

* **Cài đặt jdk:**

```
sudo apt-get install openjdk-11-jdk -y
```
* **Cài SSH và SSH Server**:
```
sudo apt-get install openssh-server openssh-client -y
```
